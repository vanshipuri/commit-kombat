# 🥊 Commit Kombat

**The Ultimate AI-Powered GitHub Roast Battle.**  
Built for developers who take their code seriously, but not their egos.

[![Live Demo](https://img.shields.io/badge/demo-live-red.svg?style=for-the-badge)](https://commit-kombat-nu.vercel.app/)

---

## 🚀 The Vibe
Commit Kombat analyzes two GitHub profiles using the **Google Gemini 1.5 Flash AI** to deliver savage, profile-specific roasts. Whether you're a "Tutorial Hell Survivor" or a "Senior Staff Janitor," the arena decides who has the most **Aura**.

## ✨ Features
- **Dynamic AI Roasting:** Real-time analysis of GitHub bios, repository names, and activity.
- **Fault-Tolerant Engine:** Includes a "Smart Fallback" system that generates profile-specific roasts even if the AI API is rate-limited.
- **Mortal Kombat UI:** 16-bit inspired design with shake effects, aura glows, and "Finish Him" sound effects.
- **Shareable Results:** One-click sharing to X (Twitter) to shame your friends.

## 🛠️ Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **AI:** Google Gemini 1.5 Flash API
- **Animations:** Framer Motion & CSS Keyframes
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
