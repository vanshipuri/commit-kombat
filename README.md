# 🥊 Commit Kombat
The AI-powered GitHub roast battle tool. Built with Next.js, Tailwind CSS, and Google Gemini AI.

## 🚀 Features
- **Dynamic Roasting:** Real-time analysis of GitHub bios and repositories.
- **Fault-Tolerant:** Includes a dynamic fallback system for 100% uptime.
- **Vibe Coding UI:** Mortal Kombat inspired design with animations and sound effects.

## 🛠️ Tech Stack
- Next.js 14 (App Router)
- Google Gemini AI (1.5 Flash)
- Framer Motion (Animations)
- Tailwind CSS
